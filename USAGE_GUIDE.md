# 営業CRMシステム 使い方ガイド

## 🎯 新機能の使い方（2025年版）

このガイドでは、最近追加された全ての機能の使い方を説明します。

---

## 📊 1. AI事前リサーチ機能

### 使い方

1. **ログイン後、「アポイント準備」メニューをクリック**
2. 見込み客リストから対象企業を選択
3. **「🤖 AI事前リサーチを生成」ボタン**をクリック
4. 数秒待つと、以下の情報が自動生成されます：
   - 📋 企業概要
   - 👤 キーパーソン情報
   - 🎯 主要な課題
   - 💡 アプローチ戦略

### 技術仕様
- **API**: OpenAI GPT-4o-mini
- **コスト**: 約$0.001-0.002/回
- **フォールバック**: APIキー未設定時はモックデータ表示

## 🔬 2. Deepリサーチ機能

### 使い方

1. **アポイント準備画面**で見込み客を選択
2. **「🔬 Deepリサーチを生成」ボタン**をクリック
3. より詳細な分析結果が表示されます：
   - 💰 財務分析
   - 🏆 競合分析
   - 📊 SWOT分析
   - 🎯 戦略提案
   - 🤝 導入シナリオ

### 活用シーン
- 重要商談の前の徹底準備
- 提案書作成の参考資料
- 経営層へのプレゼン準備

---

## 🎤 3. トークスクリプト生成

### 使い方

1. **アポイント準備画面**で見込み客と商談を選択
2. **「🎤 トークスクリプトを生成」ボタン**をクリック
3. 以下のスクリプトが自動生成されます：
   - 🎬 アイスブレイク
   - 🎯 ヒアリング質問
   - 💡 提案ポイント
   - ⚠️ 注意事項
   - 🔚 クロージング

### 特徴
- 初回アポとフォローアップで異なるスクリプト
- 企業情報と商談履歴に基づくパーソナライズ
- 実際の商談で即使用可能

---

## 📋 4. フォローアップ計画生成

### 使い方

1. 商談実施後、**アポイント準備画面**を開く
2. 該当商談の**「📝 フォローアップ計画を生成」ボタン**をクリック
3. 以下のタスクが自動生成されます：
   - 🔥 即時対応が必要（24時間以内）
   - 📅 短期フォローアップ（1週間以内）
   - 🎯 中長期アクション（1ヶ月以内）

### 活用シーン
- 商談直後のToDoリスト作成
- 次回商談の準備計画
- チーム内での役割分担

---

## 📄 5. 提案資料PDF生成

### 使い方

1. **アポイント準備画面**で見込み客を選択
2. 詳細ビューの**「📄 PDFダウンロード」ボタン**をクリック
3. 自動的にPDFファイルがダウンロードされます

### PDF内容
- 📊 企業概要
- 💡 提案内容
- 📈 導入効果
- 📅 実装スケジュール
- 💰 見積もり案内

### 技術仕様
- **ライブラリ**: pdf-lib（Cloudflare Workers互換）
- **ファイル名**: `proposal_{timestamp}.pdf`

2. 各商談の右側にある**青いPDFボタン（📄）**をクリック
3. 議事録PDFが自動生成・ダウンロードされます

### PDF内容
- 📅 商談基本情報（日時、参加者、場所）
- 📋 議題と内容
- ✅ 決定事項
- 📌 アクションアイテム
- 🔜 次回予定

## ✉️ 7. お礼メール送信

### 使い方

1. **商談履歴タイムライン**で該当商談を確認
2. **緑いメールボタン（✉️ お礼）**をクリック
3. 自動的にお礼メールが送信されます

### メール内容
- 商談実施の感謝
- 本日の商談内容サマリー
- 次のステップ（next_actionsがあれば）
- 担当者署名

### 必要条件
- 見込み客にメールアドレスが登録されていること
- Resend APIまたはSendGrid APIキーが設定されていること

## 📇 9. 名刺スキャン機能

### 使い方

1. **「名刺スキャン」メニュー**をクリック
2. **「名刺をアップロード」ボタン**をクリック
3. 名刺の画像ファイルを選択（表面・裏面可能）
4. アップロード後、**「OCR処理」ボタン**をクリック
5. 数秒で以下の情報が自動抽出されます：
   - 👤 名前
   - 🏢 会社名
   - 💼 役職
   - 📱 電話番号（固定・携帯）
   - 📧 メールアドレス
   - 🌐 ウェブサイト
   - 📍 住所

### 技術仕様
- **OCR**: Google Cloud Vision API
- **構造化抽出**: OpenAI GPT-4o-mini
- **精度**: 95%以上（日本語・英語対応）

### 抽出後の操作
- **手動修正**: OCR結果を手動で修正可能
- **人脈登録**: 「人脈に登録」ボタンで直接登録
- **見込み客登録**: 名刺情報から見込み客を作成

### 使い方

1. **「週報管理」メニュー**をクリック
2. 週報を作成・編集
3. **「📄 PDFダウンロード」ボタン**をクリック

- **ファイル名**: `weekly_report_{date}.pdf`

## 🔧 トラブルシューティング

### Q1: AI生成機能が動かない
**A:** OpenAI APIキーが設定されているか確認してください。
```bash
# ローカル環境
cat .dev.vars | grep OPENAI_API_KEY

# 本番環境
npx wrangler pages secret list --project-name sales-crm
```

### Q2: メール送信ができない
**A:** Resend APIまたはSendGrid APIキーが設定されているか確認してください。未設定の場合、コンソールにメール内容が出力されます（実際には送信されません）。

### Q3: 名刺OCRが正確ではない
**A:** 以下を確認してください：
- 画像が鮮明か（推奨: 解像度300dpi以上）
- 名刺全体が画像に収まっているか
- 文字が読みやすいか（手書きは精度が下がります）

### Q4: PDFダウンロードが失敗する
**A:** ブラウザのポップアップブロックを確認してください。また、プライベートモードでは動作しない場合があります。

### Q5: 週報PDFに日本語が表示されない
**A:** html2pdf.jsは日本語フォントに対応していますが、特殊な文字は表示されない場合があります。

## 💡 ベストプラクティス

### 商談準備の流れ
1. **前日**: AI事前リサーチを生成して概要を把握
2. **当日朝**: トークスクリプトを確認・カスタマイズ
3. **商談中**: トークスクリプトを参考に会話
4. **商談後**: 議事録を記録してPDF生成
5. **30分以内**: お礼メールを送信
6. **1時間以内**: フォローアップ計画を生成してタスク登録
7. **翌日**: フォローアップメールを送信

### データ管理
- **定期的なバックアップ**: 重要データはPDF出力して保存
- **名刺の即時スキャン**: 受け取ったらすぐにスキャンして登録
- **週報の習慣化**: 毎週金曜日にPDF出力して記録
